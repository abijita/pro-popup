== Description ==

ProPopup is a premium-quality popup plugin for WordPress that allows you to create beautiful, responsive image popups with auto-close countdown timer.

= Features =

* **Image Popups**: Upload any image for your popup
* **Auto-Close Timer**: Red background countdown timer (6,5,4... seconds)
* **Red Close Button**: Circular red close button without borders
* **Multiple Triggers**: On page load, exit intent, scroll, click
* **Display Rules**: Show on specific pages or devices
* **Device Targeting**: Desktop, mobile, or tablet only
* **Cookie Control**: Prevent repeat displays for configurable time
* **Image Links**: Make popup images clickable with URLs
* **Responsive Design**: Works perfectly on all devices
* **Beautiful Animations**: Multiple entrance and exit animations
* **Analytics**: Track impressions and performance
* **Shortcode Support**: [pro_popup id="1"]

== Installation ==

1. Upload the `pro-popup` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Popups → Add New to create your first popup
4. Upload an image and configure settings
5. Set auto-close timer and enable countdown display
6. Save and visit your site to see the popup

== Frequently Asked Questions ==

= How do I add a popup to my site? =
Create a new popup from the "Pro Popup" menu, upload an image, configure settings, and set it to active.

= Can I make the image clickable? =
Yes! There's an "Image Link" field where you can add any URL.

= How does the auto-close timer work? =
Set the number of seconds in "Auto Close" field and enable "Show Countdown Timer" to display the red countdown.

= Can I show popups only on specific pages? =
Yes! Use the "Display Rules" metabox to select specific pages or exclude certain pages.

= Is it mobile responsive? =
Yes! The popup is fully responsive and works perfectly on all devices.

= How do I track popup performance? =
Go to Pro Popup → Settings and enable analytics. You can see impressions in the popup list.
